Visualizing data

There are a few way to plot data. Either use ggplot directly (see Figure 2 for an example) or by piping (see Figure 3 for an example).

Continuous BMI is a continuous variable which often are presented as histograms.

Categorical Education is a categorical varible INERT MORE

Using ggplot raw

The code: dataset; argument - aesthetic (define look); + (“add a layer”) argument - geometry (define how data is shown)

If you’re unsure of which aesthetic you want to use: write ?geom_(aethetic) in the console

ggplot(nhanes_small, aes(x = bmi)) + 
    geom_histogram(na.rm = TRUE)
`stat_bin()` using `bins = 30`. Pick better value with `binwidth`.

Using piping to define ggplot

You don’t need to define data again since it’s piped in.

nhanes_small %>% 
    filter(bmi <= 60) %>% 
    ggplot(aes(x = bmi)) +
    geom_histogram(na.rm = TRUE)
`stat_bin()` using `bins = 30`. Pick better value with `binwidth`.
